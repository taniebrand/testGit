プレス
Microsoft AzureがクラウドオファーにRSKスマートコントラクトを追加 ジブラルタル/2019年4月16日

RIF Labs は、MicrosoftのAzure BaaSの提供に関連するRSK Smart Mainnetノードの導入が可能で、これによりMicrosoftのサービス (BaaS) マーケットプレイスでサポートされた数少ない一般向けブロックチェーンのひとつになる、と発表しました。このインテグレーションにより、分散アプリケーション (dApps) とビットコインにサポートされたサービスの展開が容易になる環境設定のための簡易プロセスが提供できます。これにより、ユーザーは、ローカルサーバーを設定し、マニュアルでノードのインストール、メンテナンスを行う代わりにRSKブロックチェーンを数分で展開、構成可能です。現在では、非ブロックチェーンとブロックチェーンディベロッパーは、RSKネットワーク上で安価かつ容易に学習および実験を行うことができます。

RIF LabsのCEO、Diego Gutierrez Zaldivarは以下のとおりコメントしています。「できるだけ簡単かつシームレスに開発を行うことはブロックチェーン技術のポテンシャルをフルに引き出すために重要なことです。MicrosoftのRSK Smart Contract Networkに対する支持は、IoVの力を一つ一つ解き放ち、大規模導入へのギャップを埋めるために大事です。」

「dAppの開発者は、ノードの設定やメンテナンスに時間をかける必要がないので、製品作りに集中することができます。 Microsoftのサポートは、RSKテクノロジーとビットコインのエコシステムの導入を加速する鍵となるでしょう」と、RSKストラテジスト兼RIF Labs CTOのエイドリアン・エイデルマンは述べています。

Microsoftは ここ数年間、R3やHyperledger Fabricといった企業用ブロックチェーンのBaaSサポートを行ってきました。しかし、ビットコインブロックチェーンのスマートコントラクトの能力を引き出す際、RIF Labsの作業継続のための本来のパートナーとして、一般向けブロックチェーンの開発をサポートする数少ないクラウドプロバイダーのひとつです。AzureのBaaSにより提供されるデベロッパーの環境により、ブロックチェーンのデベロッパーは 、迅速に、低コストでトライアンドエラーをすることができ、ブロックチェーンと従来のデベロッパーに同様に、RSKネットワークに関して、dAPPの継続した試行を極めて容易に行うことができます。

これは、RIF Labsが、オフチェーン決済、データ保存、データ提供、通信確保その他RIF OSの新しいプロトコルの実装を立ち上げるため、これから先数ヶ月はとても大事です。更に、ワンクリックのプロセスで複数のノードを設定する能力により、デベロッパーはRSKネットワークを最小限の先行費用で拡張するチャンスがあり、かつノードそのものにレンタルストレージ経由で採掘報酬を得る機会も得ることができます。ディベロッパーが、ビットコインネットワークのハッシュパワーにより確保されたイーサリアムのようなスマートコントラクト能力をますます生かせるので、これはネットワークの分散型の構造を維持する上で鍵となりでしょう。

「Microsoft Azure Marketplaceを通じて、世界中のお客様は、Azure上ですべて認証、最適化された信頼できるパートナーソリューションを簡単に見つけ、購入、展開できる」とMicrosoft社の Microsoft Azure Platformの役員であるサラン・パリハール氏は述べています。「RSKスマートコントラクトを、成長中のAzure Marketplaceのエコシステムに喜んで迎え入れたい。」

RSKSmart Orchid MainnetのAzure経由での新しいノードでの設定方法の詳細については、https://bit.ly/2VywCCv.をご参照ください。RIF Labsの詳細については、rifos.orgをご参照ください。

RIF Labsについて:

RIF Labsは、世界規模での金融包摂を可能にし、この最新技術と大規模採用の間のギャップを埋める、次世代のオープンブロックチェーンベースのインフラストラクチャの促進と開発に焦点を当てた、目的志向の組織として運営されています。

RIF Labsは、RIF OSプロトコルをオールインワンで使いやすいP2Pブロックチェーンインフラストラクチャサービススイートとして実装しています。これにより従来の開発者とブロックチェーン開発者にとって多くの拡張性が生まれ、市場投入までの時間の短縮が可能になります。

組織にはミッション志向の成長するチームがあり、ビットコインネットワークの上に展開されたスマートコントラクトネットワークの市場導入を行った、RSK Labsの創設チームに率いられた、熱心な協力者が世界中にいます。チームは、IoVに生命を与える組織のビジョンを実現するため、RIF OSや関連のプログラムといった新しいプラットフォームを通じて構築を続けます。
